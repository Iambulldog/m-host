# Mhost

Mhost is a macOS application for managing hosts, proxies, SSH, and certificates.

## Features
- Manage /etc/hosts entries
- Proxy and mkcert management
- SSH and SFTP sessions
- User-friendly SwiftUI interface

## Project Structure
- `Models/` — Data models (HostEntry, ProxySettings, etc.)
- `Services/` — Logic for hosts, SSH, proxy, certificates, etc.
- `Views/` — SwiftUI views for each feature

## Getting Started
1. Open `Mhost.xcodeproj` in Xcode
2. Build and run the app

## Requirements
- macOS
- Xcode 13+
- Swift 5.5+
